### 友情链接 配置&使用方法 --OK
**配置方法:**
创建文件 根目录/source/friends/index.md
```
---
title: friends
date:
layout: friends
---
```
创建文件 根目录/source/_data/friends.yml
```
# 单个友情链接的格式为：
name:
    link: 此处为站点链接
    descr: "介绍"
```
创建上述两个文件即可。

**使用方法:**
在根目录下_data/friends.yml中加入
```
# 样例
百度一下:
  link: https://www.baidu.com
  descr: "搜索引擎"
# 注: tab键和四个空格混用会导致报错。中英文冒号。
# 注：注意冒号:后需要空一格
```
